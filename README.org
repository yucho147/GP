* ガウス過程の簡単予測モジュール
** 概要
せっかくガウス過程を勉強したので、実装レベルで理解を目指す。
** setup
For developer
#+BEGIN_SRC bash
pip install -e .
#+END_SRC

or (for w/o dependency packages)
#+BEGIN_SRC bash
pip install -e . --no-deps
#+END_SRC
